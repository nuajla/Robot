    How to use

    1. Put the cloth in frame, then click Capture frame. This starts a new episode if none is active.
    2. Click the start point (where the robot should grab) on the image, then the end point (where it should place it). The click mode advances automatically.
    3. Click Execute. The robot runs the pick-and-place motion; the point pair is saved together with the image you clicked on.
    4. The robot returns home. Click Capture frame again to see the cloth's new position, and repeat as many times as you like.
    5. When done, click Finish episode. One more frame is captured as the final image and the whole episode (every step + final image) is written to the dataset.
